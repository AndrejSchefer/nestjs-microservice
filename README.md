## Description

Small Nestjs microservice 

## Installation

1. Install nest
```bash
$ npm install
```

2. Create .env-file
```bash
$ cp .env.examples .env
```

3. Open localhost [Localhost](http://localhost:8000/default).


## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
